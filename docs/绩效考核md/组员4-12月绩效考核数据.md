# 绩效考核评价标准

请根据计算出的【加权总分】及【团队排名】，匹配以下等级：
- **A (卓越)**：总分90分以上，排名靠前。
- **B+ (优秀)**：总分85-89分。
- **B (良好)**：总分80-84分。
- **C (合格/需改进)**：总分70-79分。
- **D (不合格)**：总分70分以下。
